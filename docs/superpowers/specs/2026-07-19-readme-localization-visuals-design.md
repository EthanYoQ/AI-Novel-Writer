# README 中文优先与首图更新设计

## 目标

恢复 v0.2.0 前的 README 正文作为内容基线，让仓库默认展示中文，并为英文 README 使用独立的英文首图；只增加 v0.2.0 更新说明与当前发布资产信息。

## 内容结构

- `README.md` 是 GitHub 默认入口，使用历史中文优先文案、中文首图和 v0.2.0 更新块。
- `README_en.md` 是英文入口，使用与中文语义对齐的英文文案、英文首图和同一份 v0.2.0 更新块。
- 两份 README 互相链接，并指向同一正式 Release。
- 不保留 v0.2.0 重写版的正文结构；不改动项目功能说明的历史含义。

## 视觉结构

- 使用两张原创的静态 PNG 首图：`hero-zh-v2.png` 与 `hero-en-v2.png`。
- 两图采用一致的暖黑、羊皮纸和克制金色编辑部写作桌视觉；只在标题语言和微小布置上区分。
- 首图是视觉处理，不作为真实产品截图；README 继续保留真实应用截图以证明产品界面。

## 验收

- 根目录 `README.md` 首图为中文 PNG，语言链接优先指向英文版本。
- `README_en.md` 首图为英文 PNG，且不引用中文首图。
- 正文来自 v0.2.0 前历史 README，只新增版本更新与 v0.2.0 下载名。
- 两张图片存在、非空、可渲染；README 图片审计通过。
